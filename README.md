# FacturApp Public Documentation

FacturApp is a private product with a public face.
This repository is the documentation layer for users, partners, and visitors who want to understand the product without seeing the source code.

## What FacturApp does

- Create and manage professional invoices
- Organize clients, companies, and products
- Export invoice PDFs
- Track invoice states from draft to paid
- Share payment links and follow up on billing
- Support bilingual EN/ES workflows
- Serve users through web, PWA, and Android TWA surfaces

## What users can expect

- Clear invoicing and billing flows
- Private source code and internal implementation
- Public-facing product explanations and policies
- A focused tool for invoicing and payment workflows, not a full accounting suite

## Policies

- We keep the code private.
- We only publish user-facing documentation.
- We do not hold client funds; payments run through Stripe.
- We do not position FacturApp as a bank, payroll system, or accounting replacement.
- We keep public content product-focused and implementation details private.

This repo keeps the public story simple:

- what the product is
